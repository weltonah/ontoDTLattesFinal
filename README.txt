---------------------------------------------------------------------------
** Se deseja utilizar a exibição de resultados via site é necessario seguir esses passos:
1) Após executar o programa copiar todas as inferencias criadas que são printadas na execução.
Os resultados são exibidos no formato ["nome1","nome2", pontos],. exemplo:
["welton", "pedro", 500],
2) Abra o arquivo ComnentDT.js presente na pasta OntoDTlattes/src/main/resources/static/resultadoGrafo
3) cole o resultado substituindo os elementos ["nome1","nome2", pontos], presentes no arquivo
4) salve o arquivo é abra o arquivo index.html presente no projeto
